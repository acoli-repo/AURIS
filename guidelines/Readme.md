# Annotation Manual

## Annotation manual 

Under `draft/` and `v.*`, you can find the AURIS guidelines for 

- the automatically supported annotation of coreference,
- the automatically supported annotation of information status ("Givenness")
- the automatically supported annotation of aspects of topichood (backward-looking center)
- the automatically supported annotation of intersentential discourse relations

For a stable version, a release with a specific version number must be created, e.g., `v.0.2/`.
The current draft version is under `draft/`.

Annotators should use the latest, non-draft version, and mark **explicitly** in the metadata which version they refer to. At the moment, this is `v.0.2`.

## Accompanying documentation

AURIS sentence-level annotations are based on ISO 24617-8 and are synchronized with related guidelines developed in collaboration with the Universitites of Porto, Portugal, and the Polish Academy of Sciences. The current draft for these can be found under `iso-background/`. Note that this should not be read by annotators, but by people developing the annotation guidelines, as definitions and examples are to be (manually) adapted for AURIS.
