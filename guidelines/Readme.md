# Annotation Manual

Annotation manual for 

- the automated annotation of referring expressions, 
- the manual annotation of coreference, and
- the manual annotation information status ("Givenness")

For a stable version, a release with a specific date must be created. 
The current draft version is under `draft/`.

Annotators should use the latest, non-draft version, and mark **explicitly** in the metadata which version they refer to.
