# Parallel subset of the GUM corpus (not integrated yet)

- based on GUM
- manual Russian translations of GUM
- restricted to files with English coref, English RST and Russian RST
- the AURIS subset is yet to be defined, based on the availability of further translations, see [files.tsv](files.tsv)
- build with

	```
	$> make
	```

 NB: for multilingual wikihow corpus, cf. https://github.com/veronica320/wikihow-GOSC?tab=readme-ov-file
